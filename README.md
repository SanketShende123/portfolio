## Sanket Shende - Developer Portfolio

Single-page dark-mode portfolio website for **Sanket Shende**, a Computer Science Engineering student and aspiring software & web developer.

### Run Locally

- **Option 1 (simple)**: open `index.html` directly in your browser.
- **Option 2 (recommended)**: serve via a local server for correct behavior of assets:

```bash
cd "Sanket Portfolio"
# with Python
python -m http.server 3000
```

Then open `http://localhost:3000` in your browser.

### Customize

- **Profile image**: place your photo at `assets/profile.jpg` (create the `assets` folder if it doesn't exist).
- **Content**: edit `index.html` to update text for About, Education, Skills, Projects, and Contact sections.
- **Colors & theme**: adjust CSS variables at the top of `styles.css` (accent color, backgrounds, etc.).
